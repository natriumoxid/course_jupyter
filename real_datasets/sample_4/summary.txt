Institute for Atmospheric Physics, University of Mainz, PANGAEA, https://doi.pangaea.de/10.1594/PANGAEA.901047 (DOI registration in progress)
Abstract:	This data set contains in-situ analysis of individual particles measured during ACLOUD in May/ June 2017. Using the single particle aerosol mass spectrometer "ALABAMA", particle size and chemical composition of single particles were studied. The data provides time, location, identified particle types, particle size (if available), flags indicating the used inlet system and a flag for cloud residual measurement mode for each analyzed particle. The particles were checked for sulfate, sodium chloride, ammonium, potassium, levoglucosan, nitrate, elemental carbon, nitrogen containing organics, trimethylamine, triethylamine and dust. A detailed description of the data processing will be done in an upcoming data paper.
